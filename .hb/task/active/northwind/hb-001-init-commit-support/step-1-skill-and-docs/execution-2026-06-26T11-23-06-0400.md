## Outcome

Before this step, `hb-init.md` step 3 delegated entirely to `committing.md` for commit message generation, but `committing.md` only documented `--task` (required) / `--step` (optional) flat-flag form. A Claude agent following `hb-init` would attempt `commit write-message-file --task <task> ...` with no task available, exiting 2. After this step, `hb-init.md` step 3 explicitly calls `plain --short "initialize hashbuild"` and `committing.md` §2 documents all three modes (`plain`, `task`, `task-step`) with a mode-selection table, per-mode required/forbidden flags, subject formats, and usage examples. Additive only — no SDK behavior changed, no other skill touched. 100 tests pass unchanged.
