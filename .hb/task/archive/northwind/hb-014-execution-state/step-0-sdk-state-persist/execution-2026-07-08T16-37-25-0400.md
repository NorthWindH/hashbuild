## Outcome

`hb-sdk` now has a durable last-executed-action record: `hb-sdk state record --skill ... --outcome ... --timestamp ...` writes `.hb/state.json`, and `hb-sdk state show [--format json|md]` reads it back (empty JSON `{}` / `"No recorded state."` when absent). `hb-sdk init` now ensures `.hb/state.json` is listed in the project-root `.gitignore`, idempotently (running `init` N times yields exactly one matching line). Manual round-trip in a fresh temp dir: `init` → `.gitignore` gained `.hb/state.json` → second `init` did not duplicate it → `state record` → `state show` returned the exact recorded fields in both JSON and markdown. Purely additive — no existing command's signature, output, or behavior changed. Full test suite green (182 passed, 0 failed, including all pre-existing suites unmodified).
